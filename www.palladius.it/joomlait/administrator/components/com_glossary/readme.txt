1. Introduction
This component is a great way to build a glossary 
on a Mambo/Joomla web site. 

 
2. Update of version 1.9.0 or higher
Always back up the database before making changes!

You may uninstall version 1.9.0 and install the 
new version.


3. Update of version < 1.9.0
Always back up the database before making changes!

The database is unchanged and to upgrade you 
should copy the new files into the same directories 
they are in at present. 

Do not uninstall an existing glossary if you want 
to retain the database entries - they will be 
*DELETED* by an uninstall.
